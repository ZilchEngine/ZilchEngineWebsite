Creating an object is a common action within Zilch Engine, in this page we outline all the ways to create one (using Sprite as an example).



 # `Create  > Create Sprite`

 ![CreateSprite](https://raw.githubusercontent.com/ZilchEngine/ZilchFiles/master/doc_files/46945.gif)

 # Open the [Search Window](https://github.com/ZilchEngine/ZilchDocs/blob/master/zilch_editor_documentation/zilchmanual/editor/editorcommands/commands.md) and type `CreateSprite`
